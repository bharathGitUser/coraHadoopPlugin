To run the program, you need to setup the Mosek system and put mosek.jar into folder "lib".
the expConstant.xml has to be placed in the folder $HADOOP_MAPRED_HOME/etc/hadoop/
To generate job profile data set, please run the /src/org/apache/hadoop/expmanager/utility/JobProfileGenerator.java
The job profile data set should be named jobSet.txt and placed at /home/hduser/log/jobSet.txt
